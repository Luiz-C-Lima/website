ARCT

I am currently involved in the development of ARCT (Arch CVE Tracker), a Python CLI program dedicated to the analysis and alerting of vulnerabilities (CVEs) in packages installed on Arch Linux-based operating systems. ARCT is designed to identify specific Arch Linux packages that have Common Vulnerabilities and Exposures (CVE) records, allowing for proactive analysis of system security. By providing CVE tracking, ARCT contributes to the protection and integrity of the operating system. This project combines Python programming skills, data management, and cybersecurity knowledge, aiming to improve the security of Arch Linux-based operating systems and ensure the reliability of installed packages.
